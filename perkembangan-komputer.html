Perkembangan Sistem Operasi Komputer

Sistem operasi adalah perangkat lunak yang paling penting untuk menjalankan komputer. Fungsinya adalah mengelola memori komputer dan proses-proses yang berjalan di komputer, serta semua perangkat lunak dan perangkat keras. Dengan demikian, Anda dapat berinteraksi dengan komputer tanpa mengetahui bahasa komputer.

Jika dilihat secara umum, sistem operasi berarti perangkat untuk mengelola semua sumber daya yang ada di sistem komputer. Selain itu, sistem operasi juga menyediakan kumpulan layanan ke pengguna untuk memberi kenyamanan dan kemudahan pengguna sekaligus pemanfaatan sumber daya sistem komputer.

Sistem operasi bisa diibaratkan sebagai sistem pemerintahan sebuah negara yang berguna untuk mengondisikan komputer supaya bisa menjalankan program dengan baik. Agar konflik antar pengguna bisa terhindari saat memakai sumber daya, maka sistem operasi akan mengatur pengguna mana yang bisa mengakses suatu sumber daya. Untuk itulah, sistem operasi sering disebut dengan sumber pembagi.

Fungsi penting dari sistem operasi yang lainnya adalah sebagai program pengendali untuk menghindari kekeliruan serta pemakaian komputer yang tidak penting. Bisa dikatakan jika sistem operasi disebut sebagai kumpulan software pengendali hardware seperti hardisk, disk drive, CD atau DVD ROM atau DVD RW, keyboard, monitor, scanner, mouse atau printer.

Generasi Pertama Tahun 1945-1955

Generasi pertama menjadi awal perkembangan sistem komputasi elektronik sebagai pengganti dari sistem komputasi mekanik. Ini disebabkan karena pencipta sistem komputer sadar jika kecepatan manusia dalam menghitung sangat terbatas.

Generasi Kedua Tahun 1955-1965

Sistem operasi generasi kedua memperkenalkan batch proccesing system yakni pekerjaan yang dikerjakan pada satu rangkaian dan dieksekusi dengan berurutan. Di generasi kedua ini, sistem operasi masih belum dilengkapi dengan sistem operasi.

Generasi Ketiga Tahun 1965-1980

Di generasi ketiga ini, sistem operasi mulai dikembangkan dengan tujuan untuk melayani banyak pemakai secara sekaligus. Di generasi ini, para pengguna sudah bisa berkomunikasi interaktif lewat terminal secara online ke komputer. Dengan begitu, sistem operasinya menjadi multi user serta multi programming.

Generasi Keempat Pasca Tahun 1980-an

Sekarang ini, sistem operasi dipakai untuk jaringan komputer dimana para penggunanya menyadari jika keberadaan komputer yang dipakai sudah saling berhubungan antara yang satu dengan yang lain. Pengguna juga sudah mendapatkan kenyamanan dengan Graphical User Interface ataui GUI yakni antamuka komputer berbasis grafis yang nyaman.

Fungsi Sistem Operasi

1. Manajemen Proses

Inti proses di dalam memori utama bertugas untuk mengarahkan pengembalian program ke dalam memori utama yang kemudian dieksekusi CPU. Kemudian, sistem operasi akan mengelola memori tersebut yakni partisi lokasi memori serta urutan data di dalam memori.

2. Manajemen Memori

Fungsi sistem operasi selanjutnya adalah untuk mengelola memori yang terdiri dari partisi per program serta virual memori. Ini dilakukan dengan cara menstimulasikan harddisk sebagai perluasan dari memori utama yang memiliki kapasitas lebih besar dibandingkan kapasitas fisik yang sebenarnya.

3. Manajemen File

Pengaturan serta pengaksesan file yang tersimpan pada penyimpanan sekunder juga merupakan tugas operasi sistem. Operasi sistem akan membuat serta mengatur struktur direktori dan beberapa operasi di file seperti delete, copy, rename serta paste.

4. User Interface

Operasi sistem menyediakan interface atau antarmuka berbasis grafis atau Graphic User Interface supaya user bisa bekerja memakai perangkat lunak yang lain.

5. Formatting

Operasi sistem juga dipakai untuk membuat format disket yang belum siap pakai sehingga nantinya bisa siap digunakan user.

6. Keamanan Data

Operasi sistem berguna untuk mengelola akses control pada file serta ketika menggunakan password.
